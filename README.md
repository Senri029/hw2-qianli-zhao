# Week 2: Build and Evaluate a Simple GenAI Workflow

## Business Workflow
I chose a customer support email drafting workflow.

## User
The user is a customer support staff member at a small online retail company.

## Input
The system receives a customer complaint or service request written in natural language.

## Output
The system produces a polite, professional, and concise draft email response in English.

## Why This Task Matters
Customer support teams often spend a lot of time writing repetitive first-pass responses. A GenAI workflow can help staff draft replies faster while still allowing human review before sending.
